  2. package filtering

    This package object defines the pre-processing algorithms introduced in Chapter 3: Data Pre-processing
    - Simple moving average for smoothing and filtering input data SimpleMovingAverage
    - Exponential moving average for filtering and smoothing data ExpMovingAverage
    - Weighted moving average for filtering WeightedMovingAverage
    - Discrete Fourier transform and series DFT
    - Discrete Fourier Low pass filter DFTFir
    - Kalman optimal filter DKalman

  3. package ga

    This package object defines the classes that support genetic algorithms as classifiers and optimizer introduced in Chapter 10: Genetic Algorithms
    - Genetic chromosome defined as a list of Gene with a symbolic representation of list of predicates (variable operator value) or rule IF( value operator value) THEN .

  4. package plots

    This package object defines the classes that wraps the jFreeChart library with default configuration parameters.

  5. package reinforcement

    This package object encapsulate the two reinforcement algorithms defined in Chapter 11 Reinforcement learning;
    - Q-learning iterative algorithm qlearning
    - Extended Learning Classifiers xcs

  6. package scalability

    This package object encapsulates application of Scalable frameworks described in Scala for Machine Learning Chapter 12 - Scalable frameworks
    - Application and performance of the Scala standard library parallel collections org.scalam.scalability.scala
    - Application of actors and futures using Akka framework org.scalam.scalability.akka
    - Introduction and description of the Apache Spark framework and MLlib library org.scalaml.scalability.spark

  7. package stats

    This package object defines the classes used in basic validation and statistics used by the machine learning algorithm presented in Scala for Machine Learning:
    - Basic statistics and normalization functions Stats
    - Validation and F1 measure for machine learning algorithms Validation
    - Illustration of the Bias -Variance trade-off BiasVarianceEmulator

  8. package supervised

    This package object defines the supervised learning algorithms described in Scala for Machine Learning:
    - Naive Bayes and Graphical models org.scalaml.supervised.bayes
    - Conditional random fields as discriminative approach to sequential data modeling org.scalaml.supervised.crf
    - Markov and Hidden Markov models as generative approach to sequential data modeling org.scalaml.supervised.hmm
    - Artificial Neural Network and Multi-layer Perceptron org.scalaml.supervised.nnet
    - Regression models including linear regression, ordinary least squares and logistic regression org.scalaml.supervised.regression
    - Kernel models and support vector machine classifiers and regression org.scalaml.supervised.svm

  10. package unsupervised

    This package object that introduces 3 key unsupervised learning methodologies:
    - K-means clustering org.scalaml.unsupervised.clustering
    - Expectation-Maximization iterative computation org.scalaml.unsupervised.em
    - Linear method for reducing dimension (size of the feature vector) using the principal components analysis org.scalaml.unsupervised.pca
