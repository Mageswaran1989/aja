org.scalaml.supervised.hmm.HMMState

DiGamma

object DiGamma

Singleton that encapsulates the computation and update of the joint probability matrix DiGamma(t, i ,j) of to be within a state S(i), transition to state S(j) at observation of index t [Formula M8]. Di-Gamma is an array of T-1 states matrices (for T observations)
