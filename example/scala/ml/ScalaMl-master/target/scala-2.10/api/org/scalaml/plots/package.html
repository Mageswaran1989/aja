package plots

This package object defines the classes that wraps the jFreeChart library with default configuration parameters. The plot supported are - Line plot for one or list of time series LinePlot
- Scatter plot for two dimension time series or two single variable time series ScatterPlot
The configuration of the plots is defined in the class PlotTheme with two basic themes:
- Plot with a black background and compatible font color BackPlotTheme
- Plot with a light grey background and compatible font color LightPlotTheme
