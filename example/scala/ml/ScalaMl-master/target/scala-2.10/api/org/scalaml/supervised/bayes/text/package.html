package text

This package object defines the application of Naive Bayes model to text mining:
- Extraction, sorting and aggregation of news articles contained in a Corpus NewsArticles
- Simple computation of relative frequency of keywords in a news article TermsScore - Computation of class likelihood and priorsNaiveBayesLikelihood
