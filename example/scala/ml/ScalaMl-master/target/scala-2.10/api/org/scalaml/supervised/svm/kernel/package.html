package kernel

This package object that different kernel functions available to the support vector machine:
- Linear kernel
- Radial basis function kernel
- Polynomial kernel
- Sigmoid kernel
- Laplace kernel

Note

Scala for Machine Learning - Chapter 8 Kernel models and support vector machines / Kernel functions.

See also

This implementation uses the LIBSVM library: http://www.csie.ntu.edu.tw/~cjlin/libsvm/
