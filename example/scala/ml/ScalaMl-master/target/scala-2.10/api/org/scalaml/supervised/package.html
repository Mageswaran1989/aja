package supervised

This package object defines the supervised learning algorithms described in Scala for Machine Learning:
- Naive Bayes and Graphical models org.scalaml.supervised.bayes
- Conditional random fields as discriminative approach to sequential data modeling org.scalaml.supervised.crf
- Markov and Hidden Markov models as generative approach to sequential data modeling org.scalaml.supervised.hmm
- Artificial Neural Network and Multi-layer Perceptron org.scalaml.supervised.nnet
- Regression models including linear regression, ordinary least squares and logistic regression org.scalaml.supervised.regression
- Kernel models and support vector machine classifiers and regression org.scalaml.supervised.svm

  2. package crf

    This package object contains the classes implementing the conditional random fields for modeling sequential data using the Indian Institute of Technology, Bombay (IITB):
    - Iterator to generate sequential data for the condition random field CrfSeqIter
    - Configuration of the conditional random fields with L2 regularization CrfConfig
    - Definition of data sequence used in training of Conditional Random fields CrfTrainingSet
    - Implementation of the Conditional Random fields using the iitb CRF library Crf
    - Definition of a CRF model CrfModel
